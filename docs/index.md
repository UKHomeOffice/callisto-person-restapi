# Person system
Callisto's person system is the master of all person-related data in Callisto. 

It ensures that changes to that data are controlled and also broadcast such that other systems within Callisto can be kept up to date

## Contents

- [features](./features/) - the features that make up person's functionality. Features link off to the containers and the parts of the containers that support them. 
- [containers](./containers/) - each container that makes up person has it's own design. These designs support the features of the person system
